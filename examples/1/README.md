# Example 1: Echo Bot

Этот пример запускает echo-бота:
- подключается к Rocket.Chat;
- подписывается на одну комнату (если задан `ROCKET_ROOM_ID`) или автообнаруживает комнаты;
- отвечает `echo: <текст сообщения>` на входящие сообщения.

## Переменные окружения

- `ROCKET_SERVER_URL` - URL сервера Rocket.Chat, например `https://chat.example.com`
- `ROCKET_USER_ID` - ID пользователя бота
- `ROCKET_AUTH_TOKEN` - токен пользователя бота
- `ROCKET_ROOM_ID` - (опционально) ID одной комнаты; если пусто, используется автообнаружение
- `ROCKET_ALLOWED_SENDER_ID` - (опционально) если задан, бот отвечает только этому отправителю

## Что проверить после запуска

- В логах есть успешное подключение/подписка.
- При сообщении в целевую комнату бот отправляет ответ с префиксом `echo:`.
- Бот не отвечает на собственные сообщения.
