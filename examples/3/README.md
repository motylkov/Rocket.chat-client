# Example 3: Auto-Subscribe Bot

Этот пример запускает бота с авто-подпиской:
- можно передать фиксированный список комнат через `ROCKET_ROOM_IDS`;
- если список пустой, бот периодически автообнаруживает доступные комнаты;
- на входящие сообщения отвечает `echo: <текст сообщения>`.

## Переменные окружения

- `ROCKET_SERVER_URL` - URL сервера Rocket.Chat, например `https://chat.example.com`
- `ROCKET_USER_ID` - ID пользователя бота
- `ROCKET_AUTH_TOKEN` - токен пользователя бота
- `ROCKET_ROOM_IDS` - (опционально) список room ID через запятую, например `id1,id2,id3`

## Что проверить после запуска

- В логах есть успешная валидация токена и подключение к WebSocket.
- Есть логи подписки на комнаты (`subscribed to room ...`).
- В периодических логах видно, что бот работает (`bot is running`).
