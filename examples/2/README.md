# Example 2: `!status` Command

Этот пример запускает бота для одной комнаты:
- подписка идет только на один room ID;
- бот отвечает `Все хорошо` только на команду `!status`.

## Переменные окружения

- `ROCKET_SERVER_URL` - URL сервера Rocket.Chat, например `https://chat.example.com`
- `ROCKET_USER_ID` - ID пользователя бота
- `ROCKET_AUTH_TOKEN` - токен пользователя бота
- `ROCKET_ROOM_ID` - ID комнаты для подписки

## Что проверить после запуска

- В логах есть `subscribed to room ...`.
- На сообщение `!status` бот отвечает `Все хорошо`.
- На любые другие сообщения бот не отвечает.
